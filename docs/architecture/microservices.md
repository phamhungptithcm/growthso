# Microservices

## Core Services

- Auth Service
- User and Organization Service
- Domain Service
- SEO Crawler Service
- Keyword and Ranking Service
- Content AI Service
- Review Service
- Ads Service
- Billing Service
- Analytics Service

## Service Boundaries

Each service owns:

- its business domain,
- data contracts,
- operational metrics,
- and deployment lifecycle.

## Communication Patterns

- Synchronous API calls for user-facing operations.
- Event-driven processing for heavy workloads.
