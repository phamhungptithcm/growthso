# SEO Intelligence Engine

## Purpose

Continuously detect technical and content-level SEO opportunities and rank them by business impact.

## Core Features

- Site audit and issue classification.
- Keyword rank tracking.
- Competitor visibility benchmarking.
- Backlink and authority trend monitoring.

## Example Output

- SEO score trend.
- Top blockers by expected traffic impact.
- Priority action list for the next sprint.

Example issue snapshot:

- SEO Score: `78 / 100`
- Missing meta descriptions on high-impression pages
- Slow page speed on key landing pages
- Duplicate headings that reduce content clarity
